Debug warn error

regreso

Uso de las directivas
  • debugg
    • lo que hace es mandar un dato a consola 
            
            body{
                @debug $colors;
                @debug $size;
            }
  • warn
    • inpeccionamos los valores de salida en la terminal, pero mandando un mensaje de alerta 
            
            body{
                @warn 'hola mundo';
            }
  • error
    • mandamos los valores a la terminal como mensaje de error. 
            
            body{
                @error 'El tipo de a variable x no es valido';
            }
Una ventaja es que no te ensucia el css

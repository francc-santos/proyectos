@extend

regreso

Nos permite crear placeholder(su usa para gurdar bloques de estilos que luego se utilizaran) y extenderlos en selectores.

Pude ser usado para definir las caracteristicas en comun de difernetes componentes etiquetas, o class y por si fuera poco tambien podemos exender directamente desde una class.
NO extender etiquetas hijas, porque es se extiende como si la nueva class fuera hijio es mejor incluso solo usar el placeholder 
    
        /* Uso de @extend */
        %button{
            display: inline-block;
            margin: 1em 0;
            padding: .5em 1.5em
        }

        /* Se llama de esa forma */
        .button{
            @extend %button
        }
        .button--alert{
            @extend %button;
            background: red;
            color: #fff;
        }
Es de lo mas comun que ua accion de hiver y active tengan el mismo estilo

Condicionales

regreso

Como en cualquier lenguaje de programacion podemos evaluar une condicion y si es que esta se cumple o no se realizara una accion dependiendo el caso.

Podemos evaluar condicionales aritmeticas, logias (== != > ) 
    
        /* Uso de condicionales */
        $dev: false !default;
        $dev: true;
        @if $dev{
            *{
                outline: 1px solid red;
            }
        }
        @if 5>3{
            content: '5 es mayor que 3';
        }@else if 3>5{
            content: '3 es mayor que 5 entonces nunca se genera';
        }@else{
            content: 'ninguna de las condiciones se cumplio';
        }

        // - == igualdad
        // - != diferencia
        // - ! negacion
        // - and para usar combinaciones de condicionales
        // - or disyuncion de condicionales , se puede usar o no () entre las condiciones.

Ciclo @for

regreso

Ciclos o iteraciones , sirven para repetir muchas veces una accion. 
     
        /* Uso de iteraciones|ciclos */

        @for $i from 0 to 10 {
            @if $i % 2 == 0{
                .column-#{$i}{
                    width: $i * 1%;
                }
            }
        }

Ciclo @each

regreso

Realizar una accion repetidamente. 
    
        /* Uso del ciclo each */
        $colors: red green blue;
        @each $color in colors {
            .button-#{$color}{
                background: #{$color};
            }
        }
        $colorsmaps : ( 
            primary: red,
            secondary: green,
            terceary: blue
        );
        @each $colormapK, $colormapV in $colorsmaps {
            .boton-#{$colormapK}{
                color : $colormapV;
            }
        }

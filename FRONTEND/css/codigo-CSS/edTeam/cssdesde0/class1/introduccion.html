Variables css

Regreso al indice

Las variables de css son para poder reutilizar propiedades de sus valores, tambien son llamadas coustom properties.

En el caso de las custome properties, se usan para estos casos donde se debe de tener un color definido para varias etiquetas o elementos , y que en caso quese quiera cambiar solo se deba de cambiar el valor de la variable. 
    
        Asi es como se declaran las variables de css 
            :root{
                --color: yellow;
            }
    
        Asi se hace la llamacion de las variables 
            h1{
                color : var(--color)
            }

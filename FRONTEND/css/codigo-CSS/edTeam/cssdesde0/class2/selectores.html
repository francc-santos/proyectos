La mejor forma de usarlo es con la identificacion de rutas

Empieza con[attr^= "value"] 
    
        a[href^="https"]{
            color : red;
        }
Es como indicar que termina con [attr$= "value"] 
    
        a[href$=".mp4"]{
            color: blueviolet;
        }
En cualquier lugar debe de contenerlo [attr*= "value"] 
    
        a[href*="team"]{
            color: royalblue;
        }

Background-color

regreso
Background es un shorthand, no es una propiedad como tal

La forma correcta de usar el propiedad background que solo cambia el color es: 
    
        background-color: red;
        /*para poner una imagen pero no es casacada porque es una porpiedad diferente*/ 
        background-image: url();

Background-attachment

regreso
Indica si el fondo va hacer scroll con el contenido o si se va a quedar fijo. 
    
        Con background attachment fixed se le dice que no tome en cuenta el scroll y permanesca en la posicion original

Background-multiples

regreso
Un elemento pude tener multiples fondos 
    
        body{
            background: url(playa.jpg) 0 0, 
                        url(carretera.jpg) center center, 
                        url(faro.jpg) right bottom, yellow;
            background-repeat: no-repeat;
            background-size: 20%;
        }

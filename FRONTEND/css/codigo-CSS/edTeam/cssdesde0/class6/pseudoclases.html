UNA PSEUDOCLASE NO PUDE SOBREESCRIBIRCE, OSEA NO EXISTE LA CASCADA CON ELLO

Se complementa con los selectores, con los que determinados de manera justa cuales el estilo que se le va a aplicar a el elemento en cuestion. Ps estas son un tipo de selector muy interesante porque son selectores de estado que se aplican de acuerdo a una circuntancia determinada no siempre.
Permite seleccionar elementos segun su estado o alguna condiccion.
Se indica con dos puntos (:) , pero como sabemso se damos un espacio entre elemento y los dos puntos esto indica que cualquier descendiente del elemnto va a tomar el comportamiento en este caso de hover.

:target :not :empty

regreso

  • :target
    • permite seleccionar un elemento destino cuando se pulsa un enlace
  • :not
    • negar una condiccion, como añadir una excepcion
  • :empty
    • detecta cuando un elemento esta vacio, osea que por ejemplo un parrafo no tenga nada de contenido u otro elemento 
        
        .secreto{
            display: none;
        }
        .secreto:target{
            display: block;
        }

  • :first-child
    • selccina el primer hijo, del mismo tipo de elemento
  • :last-child
    • selcciona el ultimo hijo, del mismo tipo de elemento
  • :first-of-type
    • no funciona de forma directa con clases , y lo que se dice es selecciona el primer hijo que sea del tipo del inicio
  • :last-of-type
    • selecciona el ultimo hijo con el mismo tipo de la etiqueta.

Ejemplo de first-child y last child

En una lista no ordenada se toma el primer hijo y el ultimo coloreando el fondo con tonos diferentes de azul, tomando en cuenta que si se da un espacio lo qu se dice es que es decediente de del elemento y si no es como si seleccionaramos desde el hijo
En el segundo ejemplo el tipo de los hijos del div cambia, por lo que podemos usar e type-of para ser mas espeficiso a que tipode etiqueta sea el primer y ultimo

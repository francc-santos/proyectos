Especificidad

Regreso al incicio

Especificidad, que hace referencia a algo muy puntual, osea que en este caso el navegador busca cual de todos es mas especificico , cual esta apuntando directamente al elemento y cual es el mas general, o tene mas class o ids referenciados.

  • General = el que envuelve muchas etiquetas
  • Especifico = envuelve menos etiquetas
La especificidad se calcula con sumas donde se le da un valor a cada tipo de selector
  • etiquetas y pseudoelementos 1
  • classes, atributos y pseudoclases 10
  • id 100
  • estilos en linea 1000
Mucha especificidad es mala practica

Herencia

Regreso al incicio

Capacidad que tienen los elementos de html de heredar los estilos de sus padres o ancestros, algo importante los estilos de las cajas no se hereda 

        body{
            color: blue ;
        }
        /* Esto hereda una gran parte de los estilos a sus hijos */
En caso de que se quisiera forzar la herencia del padre, se tiene que colocar la propiedad y en el atributo inherit 0 initial para hacer que tome el valor por default.

  • inherit
    • forzar la herencia del padre se debe de usar la propiedad y valor inherit
  • initial
    • para hacer que tome al valor por default del navegador
Un enlace por ejemplo no hereda todas las caracteriticas de su padre.

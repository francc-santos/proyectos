Propiedad display

regreso

Definir como se va a mostrar un elemento en pantalla , si es como bloque , como linea , como linea y bloque a la ves o no se va a mostrar para nada.

Propiedades de display
  • block
    • hace que todo elemento se comporte como bloque, ocupe toda la linea etc.
  • Inline
    • ps se comporta como un elemento de linea
  • Inline-block
    • uno al lado del otro , pero que se le puede dar ancho y alto, las imagenes son pro naturaleza inline-block
  • none
    • hace que no se muestre el elemento, puede ser para ocultar elementos al cambiar de tamanio de pantalla.
  • Flex
  • grid
            
                h1 {
                    display: Inline;
                    o
                    display: block;
                    o
                    display: none;
                }

Si se inspecciona el elmentose pude ver la box de las herramientas de desarrollador con la propiedad float podemos mover elementos de bloque se ballan uno al lado del otro, pero ya no se usa mucho por lo de flexbox y grid
Defininicion todos los mergenes 
    
        .box{
            margin-top: 20px
            margin-right:30px
            margin-bottom:10px
            margin-left:50px
        }
Pero se pueda hacer toda la definicion de los margenes, en una sola propiedad se le llama shorthand, y empieza a definirse en la manesillas del reloj como en el ejemplo anterior 
    
        .box{
            margin: 20px 30px 10px 50px;
        }
En caso de que falte una propiedad se usa la misma medida de acuerdo a su espejo. Tambien puden ser mediante porcentajes pero no es usual, es relativo a su padre
Para poder centra un box se puede hacer 
    
        .box{
            margin-left: auto;
            margin-right: auto;
        }
NO setear en 0 culaquier medida se usa la propiedad de auto
COLAPSADO DE MARGENES VERTICALES
Cuando hay margenes verticales se contraen , y no se suman los margenes verticales, o es como que no se puede , lo recomendale es usar bottom, o se puede usar un padding cuando se separa un hijo de un papa.

O en caso de que se quiera mover elemento conservadon la prporcion entre estos se pueden usar mergenes relativos.

Propiedad box-sizing

regreso

Es como calcula el navegador el tamanio de la caja final. A donde va a aplicar el width y el height que nosotros le estamos pasando, la caja en si termina hasta el border, para poder indicar que respete el tamanio contando el border y el padding le tenemos que decir con box-sizing : border-box pude causar que se desborde el texto 

        css 
        .box2{
            box-sizing: border-box;
        }
O podemos decir que todos los elemetos block se comporten como border-box, y asi que el tamanio indicado desde el principio se respete y no cresca con el padding y el border 

        con el uso de pseudoelementos 
        *, *:before, *:after{
            box-sizing: border-box;
        }

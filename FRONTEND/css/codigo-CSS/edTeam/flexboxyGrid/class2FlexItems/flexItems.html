Flex Items

regreso

Hola mundo
Los hijos directos del contenedor que tiene el container, solo estos se convierten en items 
    
        div id="e1">
            Hola mundo
        /div>

        #e1{
            display:flex;
            height: 200px;
            background-color: bisque;
            justify-content: center;
            align-items: center;
        }
Podemos ver que en el caso de que solo haya un texto como nieto del contenedor, este se tomara comi un item y se vera modificado por las propiedades de flex, en la teoria se dice que existe un contenedor item que ayuda a que este tenga las propiedades correspondientes como hijo directo del container.

flex: 1; podemos hacer que siga las lineas del padre

Los hijos son las cajas flexibles y tinene estas particularidades de la flexibilidad en css, los elementos que tengan la posicion ajustadas(fixed) o absoluta no van a poder ser modificadas o tomdas en cuenta como flex-items. bueno pueden ser moficcados en su posicion mediante las propiedades del papa como justify-content-content y aling-items

Tener en cuenta que los margin y los padding se respetan

Es la propiedad que permite que los flex-items crezcan , indica el factor de crecimiento, el navegador divide el espacio disponible entre la cantidad total de factores de crecimineto y se los asigna a los items 
    
        div id="e2">
            div class="e2Item">1/div>
            div class="e2Item">2/div>
            div class="e2Item">3/div>
            div class="e2Item">4/div>
            div class="e2Item">5/div>
            div class="e2Item">6/div>
            div class="e2Item">7/div>
            div class="e2Item">8/div>
            div class="e2Item">9/div>
        /div>

        .e2Item{
            background: pink;
            flex-grow: 10;
        }
        .e2Item:nth-child(even){
            background:purple;
        }
        .e2Item:nth-child(3){
            flex-grow: 3;
            /* CANTIDAD DE FLEX GROW CALCULADA POR EL NAVEGADOR 
                10
                10
                3
                10
                10
                10
                10
                10
                10
                10
                = 93 
            FACTROR DE CRECIMIENTO 
                FC = ESPACIO DISPONIBLE / 93
        
            TEniendo el factor de crecimiento, el navegador lo multiplica por el valor de flex-grow en cada caso y lo suma 
                dimencion de crecimiento * fc 
            */
        }
Los valores que acepta flex-grow son numeros enteros, decimales
Podria servir para poder hacer un sidebar que sea inamobible

Define el tamaño inicial en el eje principal, este pude ser vertical u horizontal , por lo que se pude comportar como height o como width dependiendo de la direccion que tenga en eje principal, que se modifica con flex-direction
Es el tamaño inicial, asi que en combinacion con flex-grow podeoms decir que todos los items tengan el mismo espacio sin depender de saber cuantos elementos se encuentran 
    
        div id="#e4">
            div class="e4Item">qwewrewrewrwre/div>
            div class="e4Item">qwewrewrewrwredsdds/div>
            div class="e4Item">qwewrewrewrwredsddsdsdsds/div>
        /div>

        #e4{
            display: flex;
            font-size: 2em;
            background-color: yellow;
        }
        
        .e4Item{
            background-color: violet;
            flex-grow: 1;
            flex-basis: 0;
        }
        .e4Item:nth-child(even){
            background-color: lightcoral;
        }
de esta manera podemor asegurar que todos los items ocupen el mismo espacio

Propiedad flex

regreso

qwewrewr ewrwredsdds
qwewrewrew rwredsddsdsdsds
Es le shorthand de las propiedades de flex-items en el orden de : flex-grow | flex-shrink | flex-basis

Valores frecuentes de combinaciones de nuemeros:
  • initial = 0,1,auto
  • auto = 1,1,auto
    • Crece segun su contenido
  • none = 0,0,auto
    • no crecimiento
  • n = n,0,0
    • cualquier numero flex 1 es como el ejemplo anterior 
    
        div id="e5">
            div class="e5Item">qwewre wrewrwre/div>
            div class="e5Item">qwewrewr ewrwredsdds/div>
            div class="e5Item">qwewrewrew rwredsddsdsdsds/div>
        /div>

        #e5{
            display: flex;
            font-size: 2em;
            background-color: yellow;
        }
        
        .e5Item{
            background-color: violet;
            flex: 1 1 0;
        }
        .e5Item:nth-child(even){
            background-color: lightcoral;
        }

Indica un posicion indistinta al flujo del html que hayamos declarado, por defecto la posicion de todos en la actual es 0, y para poder posicionarlo entre dos items se deben de modificar dos items como por ejemplo -2,-1 
    
        div id="e6">

            div class="e6Item">2/div>
            div class="e6Item">3/div>
            div class="e6Item">4/div>
        /div>

        #e6{
            display: flex;
            font-size: 1.2em;
            background-color: yellow;
            height: 200px;
        }
        .e6Item{
            background-color: violet;
            height: 100px;
            width: 100px;
            margin: 20px;
        }
        .e6Item:nth-child(3){
            background-color: rgb(110, 57, 57);
            order: -1;
        }
        .e6Item:first-child{
            background-color: darkgreen;
            order: -2;
        }

Main axis y cross axis

regreso

1
2
Necesita de la relacion de padres e hijos para que pueda actuar el modelo de layout.

El flexcontainer es el padre en el modelo de layout, mientras que los flex items representan a los hijos, cuanado a un elemento de se el pone la propiedad display: flex, se vuleve un flex container y los hijos en items que pudenn crecer o comprimirse en este moemento

Main y cross axis son de los conceptos mas importentes de flexbox,que el main cruza al container de manera horizontal de izquierda a derecha y el cross lo cruza de manera vertical de arriba hacia abajo , pero pueden ser cambiados 

            
                div class="flex-container">
                    div class="flex-item">1/div>
                    div class="flex-item">2/div>
                /div>
            
        
            .flex-container #e1{
                background-color: #333;
                display: flex;
                flex-direction: row-reverse;
            }
            
            .flex-item{
                background-color: #eee;
                padding: 1rem;
                margin: .4em;
            }
Los elementos se acomodan a lo largo del main axis, sin importar que compotamiento tenga este.

Al flex container se le puede poner la propiedad flex o inline-flex, donde el primero se comporta como block y el segundo como elmento de linea

La diferencia solo es para el papa

En si flex-direction determina la direcciona del main axis.
Flex-wrap te indica como los hijos se van a alinear a una (no-wrap) o varias lineas (wrap). que es modificar el comportamiento de cross-axis 
        
            .flex-container#e2{
                background-color: #333;
                display: flex;
                /* flex-direction: row-reverse;
                flex-wrap: wrap-reverse ; */
                /* shorthand */
                flex-flow: row-reverse wrap-reverse;
            }
Al final la propiedad que puede reflejar el flujo del main axis y el cross es un shorthand que es flex-flow: row-reverse wrap-reverse;

      Alineacion de items
    • justify-content
      • Alinea los items en el eje principal = main axis
      1. flex-start
        2. El valor predeterminado que se inicia desde donde se encuente el main axis
      2. flex-end
        3. Se acomoda al final, es decir emppieza desde el main-end
      3. center
        4. Ce centran respecto al eje principal
      4. space-aroud
        5. Calcula el navegador el margin
      5. space-between
        6. Es espacio que se genera es entre ellos
    • aling-items
      • Alinea los items en el eje secundario = cross axis dentro de la linea , osea con respecto al contenedor
      • Flex-start
        • Se acomadan al principio del contenedor
      • center
        • Se centran con respecto al contenedor
      • flex-end
        • Se acomoda en el final del contenedor
      • Strech
        • Ocuapan todo el height disponible del contenedor, en caso de que se tenga un height los items no se van a mover
      • Baseline
        • A la linea base del contenido
      Alineacion de lineas
    • aling-content
      • Alinea las lineas respecto al container, NO los items
      Tiene las misma propiedades del justify-content
      • flex-end
        • todas la lineas se van pa bajo

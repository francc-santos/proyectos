Que es css grid?

regreso

hasta donde lo puedo ver ahorita grid ayuda como una extension de flexbox, ya que este es un modelo de layout de una linea, si bien existe un main axis y un cross-axis estos solo sirven para alineacion, pero para layout en flexbox solo podemos usar el main axis , nunca el cross-axis. Css grid, a diferencia de flexbox permite organizar (ya no alinear) a traves de dos ejes horizontal y vertical (inline y block) No importa la posicion de un elemento en el codigo de html para construir el layout, al igual que flexbox requiere de una relacion de padres a hijos

Terminologia y elementos de CSS grid

regreso

    Elmentos que existen en el DOM

  • grid-container
    • Es el elemento al que le hayamos aplicado la propiedad
    display: grid;
    display: inline-grid;
    La diferencia entre estos dos es como se conportan respecto a su alrededor
  • grid-items

    • hijos directos
    pseudoelemntos ::before y ::after
    texto

    NO existen en el DOM, el navegador los crea para posicionar y alinear elmentos

  • grid-tracks
    • Son las filas y columnas
  • grid-lines
    • las separaciones de los tracks , se encuentran delimitando los trascks
  • grid-cells
    • Es la interseccion entre una fila y una columna
  • grid-areas
    • Cualquier area rectangular delimitada por 4 grid-lines
Podriamos definir areas y de acurdo a la colocacion de este mandar el contenido que queramos

Propiedades grid-column y grid-row

regreso

Usando el ejemplo anterior es como el css lo modifica. 
    
        .e3body{
            display: grid;
            background-color: yellow;
            min-height: 400px ;
            grid-template-columns: 1fr 3fr;
            grid-template-rows: 4em 1fr 4em;
            gap: 1em;
        }
        
        header{
            background-color: burlywood;
            /* Indican los numeros de las lineas */
            grid-row-start: 1;
            /*
            grid-column-start:  1;
            grid-column-end: 3; */
        
            /* Podemos resumirlo con el shordhand
            grid-row: grid-row-start / grid-row-end
            grid-column: grid-column-start / grid-column-end
            */
            grid-column: 1 / 3;
        
            /* O con la ayuda de span podemos indicar solamente cuantas columnas es que va a ocupar la linea anterior indica que va a ocupar desde la columna 1 a la 3, esto quiere decir que va a ocupar dos posiciones y eso se pude indicar con la etiqueta de span como: */
        
            grid-column: 1 / span 2;
        }
        main{
            background-color: cornflowerblue;
            grid-column-start: 2;
            grid-column-end: 3;
        }
        aside{
            background-color: bisque;
            grid-column-start: 1;
            grid-row-start: 2;
        }
        footer{
            background-color: darkmagenta;
            grid-row-start: 3;
            grid-column-start:1 ;
            grid-column-end:3 ;
        }

            gap: 1em;
        }
        
        header.e4{
            background-color: burlywood;

            grid-column: 1 / span 12;
        }
        main.e4{
            background-color: cornflowerblue;
            grid-row: 3;
            grid-column: span 9;
        }
        aside.e4{
            background-color: bisque;
            grid-row: 3;
            grid-column: 1 / span ;
        }
        footer.e4{

        }
        
        .feature{

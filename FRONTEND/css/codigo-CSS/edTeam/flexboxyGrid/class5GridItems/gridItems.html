tracks flexibles con minmax(), auto-fit y auto-fill

regreso

La funcion minmax(), sirve para poner un maximo y un minimo pero en este caso puede que se desvorde ya que no se respeta el contenido

En el caso de auto-fill y auto-fit, lo que decimos es crea tanatas columnas o filas como puedas de este tamaño 
    
        .e3Container{
            background-color: darkmagenta;
            padding: 1em;
            display: grid;
            gap: 1em;
            /* grid-template-columns: min-content min-content; */
            /* grid-template-columns: min-content max-content; */
            grid-template-columns: repeat(auto-fill, minmax(200px,1fr))
        }
        .e2Item{
            background-color: darkviolet;
        }
        
        .e3Item img{
            width: 200px;
        }

Cualqiuer espacio rectancular dividido por lineas de css grid es una area, estas se puden definir con la propiedad grid-template-areas 
    
        .e4Body{
            font-size: 2em;
            min-height: 500px;
            display: grid;
            gap: 1em;
        }
        .e4Item{
            background-color: blueviolet;
        }
        @media screen and (min-width: 1024px) {
            .e4Body{
                grid-template-areas: 'header header'
                                     'aside main'
                                     'footer footer';
                grid-template-rows: 2em 1fr 2em;
                grid-template-columns: 1fr 3fr;
            
            }
        
            header{
                grid-area: header;
            }
            aside{
                grid-area: aside;
            }
            main{
                grid-area: main;
            }
            footer{
                grid-area: footer;
            }
        }

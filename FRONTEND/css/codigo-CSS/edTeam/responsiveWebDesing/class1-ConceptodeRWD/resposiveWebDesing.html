Concepto

regreso

Diseño web adaptable, es una metodologia, de patrones de diseño que lo que buscan es que una web se vae de menera correcta en cualquier tipo de dispositivo .

Que la experiencia de usuario sea correcta para cualquier usuario en culquier cliente.

Es necesario conocer las bases para poder usar frameworks de maenera responsable.
El diseño web resposivo segun un primer articulo se basa en:
  • las grillas resposivas
  • las imagenes resposivas
  • las media queries
Appel introdujo meta:viewport, sin neceaidad de crear una pagina para cada dispositivo.

Viewport

regreso

El viewport es toda la parte visible de nuestro navegador para mostrar el contenido de la web. y la barra de scroll con es parte del viewport.
Tipo de meta que inventa apple, que puede tener diferentes caracteristicas como: meta name="viewport" content=""
  • width=device-width
  • user-scalable=no|yes
    • prohibir el zoom
  • initial-scale=1
    • scale inicial
  • maximun-scale
    • zoom maximo que puede dar el usuario
viewport!=resolucion de la pantalla es decir que 1px de wiewport es diferente que 1px de pantalla

Media Queries

regreso

En la traduccion , podemos dcecir que es una consulta de medio, donde se pregunta en que medio se esta consultando la web, pero tambien si conocemos que en un caso una parte de la web sera en formato impreso debemos de saber cuando esto ocurre.

La forma en que realizamos estas cnosultas es: @media screen| print, pero esto no vale de nada al menos en el diseño web, porque todo se visualizara en una pantalla.
  • min-width= que cumpla con un minimo de ancho
  • max-width= que cumpla con un maximo de ancho
  • orientarion: landscape = que el ancho sea mas grande que el alto
  • orientarion: portraint = que el alto sea mas grande que el ancho
  • min-aspect-ratio = que cumpla con el minimo de na escala dicha
  • divice-aspect-ratio= 
    
        body{
            background-color: darkblue;
        }
        
        @media screen and (min-width:600px){
            body{
                background-color: darkmagenta;
            }
        }
        
        @media screen and (max-width:600px){
            body{
                background-color: darkmagenta;
            }
        }
        
        /* Otro tipo de media queries */
        @media screen and (orientation: landscape) {
            body{
                background-color: darkorange;
            }
        }
        @media screen and (orientation: portrait) {
            body{
                background-color: darkorange;
            }
        }
        @media screen and (min-aspect-ratio) {
            body{
                background-color: darkorange;
            }
        }

Metodologias

regreso

  • Mobile first
    • empiezas diseñando para un mobil y en ves de ir quitando funcionalidades en el main de lo visual se van añadiendo.
  • Desktop first
    • Es practicamente lo contrario a mobile first, en este caso vas ocultando las funcionalidades por ejemplo en un sidebar, lo malo de esto es que en ocaciones se pueden olvidar ciertas cosas y quedar to mal, por eso es mejor mobile fisrt
  • Content first
    • Fuerza en user experience, tomando muy en cuanta como es que se deforma la maquetacion con mas o menos contenido que sea real.

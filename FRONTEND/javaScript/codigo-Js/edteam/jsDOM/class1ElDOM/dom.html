Nodos

regreso

Es el nombre correcto para referirse a los elemento del DOM, recordar que el DOM no es parte de la especificacion de JavaScript, sino que es una API web que se consume con JavaScript, es similar a hablar del local storage, canvas, geolocalizacion, multimedia, no son parte tecnicamente del lenguaje en si. Entonces un nodo es cada etiqueta html, asi como los atributos, el texto., es como un arbol que se va ramificando, donde cada que se remfica es un nodo. LOs nodos en el navegador se pueden ver con la propiedad nodetype que todos los nodos la tienen. 
        
            .nodetype
            .nodeName /*Este es keysensitive usamos para verificar */
Los nodos que generalmente nos van a interesar son element-node, text-node, coment-node TODA etiqueta HTML es un ELEMENTO., asi que al ejecutar el nodetype, devuelve un 1 = elemento, 2= atributo, 3= texto, 8= comentario

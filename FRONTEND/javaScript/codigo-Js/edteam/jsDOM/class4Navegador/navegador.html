Objeto Date

regreso

Es un objeto que trae las fechas desde el sistema operativo, asi que si se cambia la fecha desde el sistema operativo este cambiara.
  • Instanciar
    • new Date()
      • Como e un objeto se debe de intanciar
    • new Date(year, month, day)
      • month = desde 0 hasta 11
    • new Date(year, month, day, hours, minutes, seconds)
      • igual que la anterior pero siendo un poco mas especifico
    • new Date(string)
      • podemos obtener una fecha desde al declaracion de un string y con todo el la zona horaria
  • Metodos
    • getFullYear()
      • devuelve el a単o
    • getMinutes()
      • devuelve los minutos
    • getSeconds()
      • devulve los segundos
    • getDay()
      • devulve los dias de la semana desde 0 a 6
    • getDate()
      • devuelve el dia desde el 0 al 31
    • getMonth()
      • devulve el mes desde 0 a 11
    • getTime()
      • devulve los milisegundos desde el 1 de enero de 1970
    • getTimezoneOffset()
      • devulve la diferencia horaria de nuestra zona con el meridiano 0, en minutos, tabajar con las horas internacionales.

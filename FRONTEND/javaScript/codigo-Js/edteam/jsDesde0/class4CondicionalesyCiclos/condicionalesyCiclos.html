Los condicionales

regreso

Son sentencias que si se cumplen pasa algo y en el caso que no se cumpla no pasa nada o pasa otra cosa diferente. 
        
        if(condicion){
            accion
        }
Hablando de codigo podemos ver que en caso de que la condicion no se cumpla, simplemente js ignora el fragmento de codigo que esta encerrado en la condicion.

If Else

regreso

El condicional if-else considera un caso por default en caso de que la primera condicion no se cumpla, esto pude se omitir y ademas nos encontramos en una funcion porque el flujo del programa termina en la respuesta por default, o creo que mejor aun es evaluar opcion donde no ocurre nada o mucho para asi ahorrar tiempo de ejecucion y mas si estamos dentro de una funcion. 
    
        persona = "io"
        edad = 28
        
        if(edad >=18 && edad<=65){
            console.log(`${persona} puedes venir a la fiesta`);
        }else{
            console.log(`${persona} no puede venir a la fiesta`);
        }
Con la reasignacion de las variables podemmos ver un caso donde no se cumple la condicion entonces podemos dar una respuesta al usuario que no cumplio.

Switch

regreso

Permite avaluar multiples condicionales y a partir de ellas determinar la instruccion o bloque de instrucciones a ejecutar, simplificando la estructura que podria ser complicada con el else if. Porque cuando retites algo mucho en al codigo seguremente estamos haciendo algo mal. 
    
    let personalidad = "Pequeño y debil"

    switch(personalidad){
        case "Fuerte y comelon":
            console.log("Eres goku");
            break
        case "Veloz y egoista":
            console.log("Eres veggeta");
            break
        case "Pequeño y debil":
            console.log("Eres krilin");
            break
        default:
            console.log("Eres una sabandija");
    }

Ciclo while

regreso

Mientras una condicion, se hace algo , en el caso de que ya no se cumpla la condicion ya no entra a mientras. 
    
    let j=0
    while(j<=10){
        console.log(`Como no hota la tabla del 8 en ${j} x 8 = ${j*8}`);
        j++
    }

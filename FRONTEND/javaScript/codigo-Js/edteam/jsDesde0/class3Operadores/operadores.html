Type coercion

regreso

Es la capacidad que tiene el lenguaje de programacion de poder convertir cualquier dato en otro tipo de dato. 
        
    let a = 521
    let b = "422"
    let respuesta = a+b

    console.log(`Es eejemplo de typecoercion ${respuesta}`);
    console.log(`Es eejemplo de typecoercion ${typeof respuesta}`);/* string */
En este ejemplo podemos ver que al tener dos tipos de datos diferentes, para js a al hora de computar lo que importa es el operador que se esta usando porque sirve tanto para una suma como para concatenacion. Esto es como sirve para concatenacion, io js no me importa el tipo de dato que tenga siempre voy a superponer las cadenas de texto.

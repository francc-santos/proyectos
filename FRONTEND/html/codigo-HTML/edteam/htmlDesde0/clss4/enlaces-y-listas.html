Uso de las rutas relativas y absolutas

Nos permite saber como encontrar otro recuso diferente a la pagina web donde estamos
  • Ruta Aboluta
    • Tienen el protocolo http o https y son unicas como un URL; se usan para rutas externas, destinos fuera del proyecto
  • Ruta Relativa
    • Rutas internas del proyecto, por loq ue no usan protocolo y se debe de navegar como se menciona:
    1. Si el recurso esta en LA MISMA CARPETA, se escribe nombre y extension
    2. Si el recurso esta un caprte porn encima
    3. Se contruyen a partir del origen, es decir desde donde nos encontramos ubicados
  • Ruta Relativa a la raiz
    • Se contruyen a partir de la carpeta raiz del proyecto , comom en linux / representa la raiz del proyecto.
    Un estandar es usar las rutas del proyecto como relativas desde la raiz, pq te muesta como esta formado el proyecto.

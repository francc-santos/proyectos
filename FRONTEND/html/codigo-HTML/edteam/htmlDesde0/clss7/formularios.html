Estructura basica de un formulario

regreso

En las demas etiquetas marcamos el contenido, pero en el formulario lo que marcamos son elementos para que los usuarios interactuen con la pagina web, de los componentes mas importantes de las paginas web, pagos , registros , compras de un vuelo, etc.
El atributo name dentro de los formularios es como le decimos al backend que el ususarioa llenado los campos necesarios para poder ser enviados. 
    
        !-- El atributo action dice hacia donde vamos a enviar la informacion del formulario , son enviados al servidor , este evalua que sean validos y seguros, con ello sabe que responder -->
        form action="form.html">
            !-- campos input sin el type es como el tipo text -->
            div>
                input placeholder="Tu nombre de usuario" name="username">
            /div>
            div>
                input type="text" placeholder="Tu contrasenia" name="password">
            /div>
            div>
                !-- Permite un texto largo -->
                textarea name="message" id="" cols="30" rows="10" placeholder="Dejanos un mensaje">/textarea>
            /div>
            !--  Abajo otra forma de enviar los datos del fomulario-->
            !-- input type="submit" value="ENviar"> -->
            button>Enviar/button>

        /form>

Estructura completa de un formulario

regreso

Placeholder no se puede traducir y no puede controlarlo, por lo que se usa la etiqueta lable que por medio de un id en el input que le indiquemos actua como le palce holder.
Otra forma de hacerlo , es que podemos anidar el input dentro del lable y olvidar el id y el for 
    
        form action="form.html">
            !-- campos input sin el type es como el tipo text -->
            div>
                label for="username">Tu nombre/label>
                input id="username" name="username">
            /div>
            div>
                label for="password">Tu contrasenia/label>
                input type="text" id="password" name="password">
            /div>
            div>
                !-- Permite un texto largo -->
                label for="message">Dejanos un mensaje/label>
                textarea name="message" id="message" cols="30" rows="10" placeholder="Dejanos un mensaje">/textarea>
            /div>
            !--  Abajo otra forma de enviar los datos del fomulario-->
            !-- input type="submit" value="ENviar"> -->
            button>Enviar/button>
        /form>
VALUE podemos darle un valor predeterminado, y fieldset nos permitte agrupar los campos y con legend podemos etiquetear el conjunto

Atributos en los formularios

regreso

  • placeholder
    • es el del texto que te indica que valor se esta esperando por parte del usuario
  • type
    • que tipo de input estas trabajando
  • require
    • indica que los atributos son necesarios
  • Readonly
    • no pude ser editado por el usuario
  • autocomplete
    • on/off solo en el form, que si el navegador ya tiene los valores guardados podemos tomarlos y llenar los campos del form
  • checked(checkbox y radio) y selected (select)
    • indicamos un valor por default

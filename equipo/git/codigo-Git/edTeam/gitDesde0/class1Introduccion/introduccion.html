regreso

Las bitacoras de capitan son donde se apuntan los sucesos que van sucediendo en torno al viaje, en secuencia cronologica y podemos ver todos los logs que ha sucecido en el viaje.

Principalmente sirve para
  • Describir todos los detalles del viaje
  • Recirdar un aspecto especifico
  • Poder ense√±ar los detalles del viaje a otras personas
  • Es nuestra experiencia docuementada
Al estar documentado podemos retomar errores que hemos cometido y poder usarlos para resolver otros parecidos.

Introduccion a git

regreso

Sistema de control de versiones

Una version es una iteracion de una proyecto con el fin de mejorarlo., osea la suma de nuevas caracteristicas como interface, funciones del so, cambio de optimizacion de recursos, seguridad, etc
Controlar las verisones es tener en cuenta los cambios que se han dado entre estas versiones
Git es creado por linus torbal, UN Repositorio es como una caja donde se guardan los cambios(folders), este repo tendra todo el historial de lo que se ha hecho en el proyecto

Podemos tener los estados.
  • Espacio de trabajo

  • Area de preparacion
    • Punto intermedio donde le decimios a git que le estamos pasando ciertos archivos., osea podemos describir las diferentes partes de lo que se ha modificado de los archivos en cuanto al proyecto

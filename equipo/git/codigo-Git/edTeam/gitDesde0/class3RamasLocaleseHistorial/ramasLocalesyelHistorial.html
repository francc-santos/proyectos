Creando ramas locales

regreso

Recirdar mirar donde se encuentar nuestro head que indica el nodo (luagar ) en el tiempo donde nos encontramos actualmente. 
    
        git checkout -b nombre_nueva_rama
        Se crea una nueva rama con el nmombre: nombre_nueva_rama

        git branch
        vemos las ramas que tenemos creadas y en cual rama no encontramos actualmente

        git log --all
        toda la info de las ramas


        git log --oneline --graph --decorate
        tipo arbol de als rammas y con decoraciones 

        git checkout master
        cambiamos a la rama master

        git switch master
        cambio a master  a partir de la version 2.23
